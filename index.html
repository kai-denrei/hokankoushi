Neural Sheet

A faux self-organizing map — all interpolation, zero learning.

A flat lattice folds onto a 3-D shape while sparks of “activation” roam and shrink, mimicking a SOM discovering the structure of data. Nothing is trained: every frame is pure arithmetic into one shared position buffer.

The fold

Each of the 64×64 vertices moves along a straight line from its flat position to its spot on the target shape, by a per-vertex eased progress s:

p(u,v,t) = mix( flat , target , s(u,v,t) ) + wobble

A per-vertex stagger offset d delays each vertex so a visible front crosses the sheet instead of everything tweening at once:

raw = clamp( (t/dur − d·spread) / (1−spread) , 0, 1 ) s = ease(raw)

d(u,v) blends radial distance from a seed point with hand-rolled value noise.

Annealing overshoot

The easing adds a decaying sine so each region overshoots and settles — it reads as physical annealing, not a CSS transition:

ease(x) = easeInOutQuint(x) + amp·sin(x·π·freq)·(1−x)^decay

Wobble

Ambient life displaced along the interpolated surface normal (fabric, not jelly), fading as the fold completes:

wobble = n̂ · amp · (1−gp)^decay · sin(t·freq + φ)

Sparkles — the SOM tell

Fake best-matching-unit activations. Per-vertex brightness sums Gaussian neighborhoods whose radius σ shrinks across the run — the signature of a contracting SOM neighborhood:

b = Σ exp( −gridDist² / 2σ² ) · envelope σ = mix( σ_start , σ_end , gp ) // wide → narrow

Shapes & blend

Every target is a pure function (u,v) ∈ [0,1]² → ℝ³, normalized to a unit sphere so framing is shape-independent. Torus, swiss roll, Möbius, pyramid & bipyramid, two parametric families you can sculpt with sliders — the superformula and spherical harmonics — and a tesseract (nested 4-cube cells rotated in 4-D and perspective-projected). The folded surface is a live blend, and the loop tours through shapes. Node colour can ramp through scientific colormaps (viridis / inferno / turbo):

const s = ease(clamp((t/dur - d*spread)/(1-spread),0,1));
const g = mix(shapeA, shapeB, blend);   // A → B morph
base    = flat + (g - flat) * s;        // the fold
pos     = base + normal * wobble;       // ambient life

Interaction

The pointer lights nearby nodes (a screen-space Gaussian), clicks drop damped radial waves, and the camera parallaxes toward the cursor:

ripple = amp · exp( −(gridDist − speed·age)² / 2w² ) · e^(−age/τ)

Every value here is one field in a single CONFIG object — the gear (top-right) edits them live.
